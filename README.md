- `Windows Terminal`文件夹中，包含`Windows Terminal`的配置文件：`setting.json`，该文件应放在`C:\Users\<username>\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState`中。
